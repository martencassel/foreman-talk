
# Active Directory Server Setup

## 1. Install Windows Server 2025 from ISO

Standard installation process.

***

## 2. Trigger Windows Update

```powershell
wuauclt /detectnow
```

***

## 3. Disable Windows Firewall

```powershell
Set-NetFirewallProfile -Profile Domain,Private,Public -Enabled False
```

***

## 4. Set Static IP Address and DNS

```powershell
Get-NetAdapter | Where-Object Status -eq Up | New-NetIPAddress -IPAddress 192.168.0.50 -PrefixLength 24 -DefaultGateway 192.168.0.1
Get-NetAdapter | Where-Object Status -eq Up | Set-DnsClientServerAddress -ServerAddresses "8.8.8.8","1.1.1.1"
```

***

## 5. Enable OpenSSH Server

```powershell
Add-WindowsCapability -online -Name "OpenSSH.Server~~~~0.0.1.0"
Start-Service sshd
```
